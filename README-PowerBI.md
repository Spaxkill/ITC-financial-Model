# ITC Ltd. — Power BI Dashboard

**Overview**
This project contains a Power BI dashboard built on top of a financial model for **ITC Ltd.** The dashboard provides interactive visualizations of historical and projected financials, key ratios, and valuation metrics. It allows users to explore company performance and financial insights through dynamic visuals and drill-through analysis.

**Files in this repository**
- `itc powerbi DB.pbix` — Power BI dashboard file.
- `README-PowerBI.md` — This file (project description).

**Key Features**
- Visualization of historical financial statements (Income Statement, Balance Sheet, Cash Flow).
- Interactive dashboards summarizing profitability, liquidity, leverage, and efficiency ratios.
- Valuation summary with DCF-based insights.
- Drill-through capability to explore detailed financial breakdowns.
- Bookmarks for switching between different analytical views and scenarios.

**How to Use**
1. Open the `.pbix` file using Microsoft Power BI Desktop.
2. Navigate through the dashboard tabs to view different analyses (historical, ratios, valuation, etc.).
3. Use slicers, filters, and drill-through to explore the data in more depth.
4. Switch between different perspectives using the pre-configured bookmarks.

**Notes**
- Data is based on the structured Excel financial model of ITC Ltd. (2016–2025).
- The dashboard is designed for analysis and educational purposes only and is not professional investment advice.
